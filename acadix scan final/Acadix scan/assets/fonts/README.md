# Fonts Directory

This directory is for custom fonts used by the Acadix Scan application.

To add custom fonts:
1. Place TTF or OTF font files in this directory
2. The application will automatically load them at startup

Recommended fonts for modern UI:
- Segoe UI (Regular, Light, Bold)
- Roboto
- Open Sans
- Montserrat

These fonts will enhance the visual appeal of the application and provide a more modern look and feel.
